# 🌱 Calculadora de Crédito de Carbono - APS 2024

## 📖 Sobre o Projeto

A **Calculadora de Crédito de Carbono** é um projeto acadêmico desenvolvido para auxiliar no cálculo estimado da emissão e compensação de carbono, promovendo conscientização ambiental e sustentabilidade.

O sistema permite realizar cálculos relacionados ao impacto ambiental de determinadas atividades, apresentando resultados de forma simples e intuitiva.

---

## 🎯 Objetivo

O objetivo do projeto é aplicar conceitos de:

* Programação
* Desenvolvimento Web
* Sustentabilidade
* Cálculo de emissões de carbono
* Interface gráfica e experiência do usuário

Além de incentivar práticas sustentáveis através da tecnologia.
